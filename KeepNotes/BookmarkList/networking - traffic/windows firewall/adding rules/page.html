You can add a rule with:
netsh advfirewall firewall add rule

Common flags are
name=""
dir=
action=
program=""
protocol=
enable=
localport=


Allowing a program like:
netsh advfirewall firweall add rule name="allow_my_program" dir=in action=allow program="evil.exe" enable=yes


Allowing ports like:
netsh advfirewall firewall add rule name="allow_my_port_80" dir=in action=allow protocol=TCP localport=80


Powershell also makes things very easy, with similar commands:
New-NetFirewallRule -DisplayName <name> -RemoteAddress <ip_address> -Direction inbound -Action allow
